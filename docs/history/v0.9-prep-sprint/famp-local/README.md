# famp-local - frozen v0.9 prep-sprint scaffolding

This script is **frozen**, not maintained. It embodies the v0.9 prep-sprint
UX validation (T1-T9) - every workaround, every `update_zprofile_init` quirk,
every flag encodes a lesson learned during the v0.8.x -> v0.9 migration design.

Bug fixes ship via the live `famp` binary or the `famp-local hook` subcommand
(Phase 2 / HOOK-04a). Do not edit this file.

See [`docs/MIGRATION-v0.8-to-v0.9.md`](../../../MIGRATION-v0.8-to-v0.9.md) for
the user-facing migration path, and ROADMAP backlog 999.6
(`update_zprofile_init` should sandbox on non-default `FAMP_LOCAL_ROOT`) for
the one open issue against this archived script.
