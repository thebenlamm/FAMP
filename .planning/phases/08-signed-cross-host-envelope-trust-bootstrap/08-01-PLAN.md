<objective>
Extend the single FAMP wire envelope with the seven forward-compatible federation
fields (sender/receiver domain, sender key_id, nonce, expiry, plus reserved
capability/approval) as optional, omit-when-empty additions covered by the one
existing INV-10 signature. This delivers the WIRE-02 schema and its byte-exact
canonical-JSON round-trip, while keeping the crypto-dropped local-bus wire
byte-identical to today (D-02).

Purpose: the cross-host envelope (Phase 9 populates + signs it over the live
transport) must be able to carry v1.1/v2.0 fields without a wire break. This
phase lands the schema and proves the round-trip; it does not send anything.
Output: extended WireEnvelope/UnsignedEnvelope with builders, plus two
regression tests (federation round-trip + local byte-identity).
</objective>
