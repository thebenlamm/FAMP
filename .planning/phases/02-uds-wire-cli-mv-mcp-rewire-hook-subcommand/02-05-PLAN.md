    `run_at(sock: &Path, args: ListArgs, mut out: impl std::io::Write)`:
    1. Resolve identity via `resolve_identity(args.send_as.as_deref())?`.
    2. Compute `since`: if `args.since.is_some()` use that; else read `<bus_dir>/mailboxes/.<identity>.cursor` via `famp_inbox::InboxCursor::read()` (default 0 if absent).
    3. `let mut bus = BusClient::connect(sock).await.map_err(|_| CliError::BrokerUnreachable)?;`
    4. `let reply = bus.send_recv(BusMessage::Inbox { since, include_terminal: args.include_terminal }).await?;`
    5. Match `reply`:
       - `BusReply::InboxOk { lines, next_offset }` → for each `line` in `lines`, `out.write_all(line)?; out.write_all(b"\n")?;` (raw bytes already canonical-JSON per D-09; just append newlines for JSONL framing). After all lines, write a footer line: `writeln!(out, "{{\"next_offset\":{next_offset}}}")?;` so the user can pipe to `famp inbox ack --offset $(... | tail -1 | jq .next_offset)`.
       - `BusReply::Err { kind: NotRegistered, .. }` → `Err(CliError::NotRegisteredHint { name: identity })`.
       - `BusReply::Err { kind, message }` → `Err(CliError::BusError { kind, message })`.
       - any other reply → `Err(CliError::Io { ... unexpected })`.

    `pub async fn run(args: ListArgs) -> Result<(), CliError>` calls `run_at(&resolve_sock_path(), args, std::io::stdout().lock())`.

    ```rust
    #[derive(clap::Args, Debug)]
    pub struct AckArgs {
        /// Cursor offset to advance to. Required; obtain from `famp inbox list` next_offset.
        #[arg(long)]
        pub offset: u64,
        /// Override identity (D-01).
        #[arg(long = "as")]
        pub send_as: Option<String>,
    }

    pub async fn run(args: AckArgs) -> Result<(), CliError> {
        let identity = crate::cli::identity::resolve_identity(args.send_as.as_deref())?;
        let sock = crate::bus_client::resolve_sock_path();
        let bus_dir = crate::bus_client::bus_dir(&sock);
        // CRITICAL: NO BusClient::connect call here. The cursor is local.
        crate::cli::broker::cursor_exec::execute_advance_cursor(bus_dir, &identity, args.offset)
            .await
            .map_err(|e| CliError::Io { path: bus_dir.to_path_buf(), source: e })?;
        println!("{}", serde_json::json!({ "acked": true, "offset": args.offset }));
        Ok(())
    }
    ```

There is no implicit "ack everything I just read" mode in v0.9.
