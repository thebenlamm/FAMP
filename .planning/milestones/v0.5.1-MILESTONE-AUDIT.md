## 6. Breaking Changes Introduced (Informational)

The spec fork is wire-incompatible with v0.5 by design. Implementations conforming to v0.5 cannot interop with v0.5.1 without rework. Key incompatibilities:

1. **Agent Card:** `signature` field removed; `federation_credential` + `federation_signature` added (§6.1, Δ11/Δ12).
2. **Envelope signing:** domain-separation prefix `FAMP-sig-v1\0` now required; raw Ed25519 signature over canonical JSON is no longer valid (§7.1a, Δ08).
3. **Canonical JSON:** implementations must use RFC 8785 JCS verbatim; `serde_json` `arbitrary_precision`/`preserve_order` features are non-conformant (§4a, Δ04/Δ05).
4. **Terminal status crystallization:** ack disposition no longer crystallizes FSM state (§9.6a, Δ16).
5. **Capability snapshot:** bound at commit time, not contract time (§11.2a, Δ23).

All breaking changes are cited in the Changelog (Δ01..Δ28) with reviewer finding references.
