## Tech Debt — RESOLVED 2026-04-13

### 1. Format drift — RESOLVED
Ran `cargo fmt --all` — applied to both `famp-crypto/tests/sha256_vectors.rs` and `famp-crypto/tests/worked_example.rs`, plus reformatted the new attribute blocks in famp-canonical tests. `just fmt-check` now green.

### 2. Test-files clippy hygiene — RESOLVED
Added file-level inner attribute to all 5 famp-canonical integration test files (`artifact_id.rs`, `conformance.rs`, `duplicate_keys.rs`, `float_corpus.rs`, `utf16_supplementary.rs`):

```rust
#![allow(
    unused_crate_dependencies,
    clippy::unwrap_used,
    clippy::expect_used,
    clippy::pedantic
)]
```

Rationale: integration tests each compile as independent crates with only partial use of workspace deps (triggering `unused_crate_dependencies`), and intentionally panic on assertion failure (triggering `unwrap_used`/`expect_used`). `clippy::pedantic` relaxation covers incidental `doc_markdown`, `uninlined_format_args`, and bare-URL lints in test docstrings. Production code remains under workspace-strict lints. Closes the carried TODO from Plan 01-02.

`just lint` now green. `just ci` runs clean end-to-end.
