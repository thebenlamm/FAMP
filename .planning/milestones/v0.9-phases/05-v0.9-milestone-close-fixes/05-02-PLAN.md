<objective>
Close milestone gap #2 (PARTIAL): the HOOK-04a registration surface
(`scripts/famp-local hook add`) honors `FAMP_LOCAL_ROOT` and writes to
`${FAMP_LOCAL_ROOT:-$HOME/.famp-local}/hooks.tsv`, but the HOOK-04b execution
runner (`crates/famp/assets/hook-runner.sh`) hardcodes
`${HOME}/.famp-local/hooks.tsv` at line 9. Default path agrees; non-default
`FAMP_LOCAL_ROOT` silently breaks (registration succeeds, hooks never fire).

Purpose: Achieve full path parity between writer and reader so the HOOK-04
contract is wired end-to-end regardless of where the user roots their FAMP
local state. This is the fix the v0.9 milestone audit recommended:
"parameterize hook-runner.sh to read `${FAMP_LOCAL_ROOT:-$HOME/.famp-local}/hooks.tsv`".

Output: A repaired `hook-runner.sh` that reads the same env-overridable path
as the registration writer, plus a Rust integration test that locks the
behavior in place.
</objective>

      use std::fs;
      use std::os::unix::fs::PermissionsExt;
      use std::process::Stdio;
      use tempfile::TempDir;

      const HOOK_RUNNER: &str =
          concat!(env!("CARGO_MANIFEST_DIR"), "/assets/hook-runner.sh");

      fn write_stub_famp(dir: &std::path::Path, sentinel: &std::path::Path) {
          let bin = dir.join("famp");
          fs::write(
              &bin,
              format!(
                  "#!/usr/bin/env bash\nprintf '%s\\n' \"$@\" >> {}\nexit 0\n",
                  sentinel.display()
              ),
          ).unwrap();
          let mut p = fs::metadata(&bin).unwrap().permissions();
          p.set_mode(0o755);
          fs::set_permissions(&bin, p).unwrap();
      }

      fn synth_transcript(dir: &std::path::Path, edited: &str) -> std::path::PathBuf {
          let p = dir.join("transcript.jsonl");
          // user turn boundary, then an assistant tool_use block:
          let body = format!(
              "{}\n{}\n",
              r#"{"message":{"role":"user","content":"go"}}"#,
              format!(
                  r#"{{"message":{{"role":"assistant","content":[{{"type":"tool_use","name":"Edit","input":{{"file_path":"{}"}}}}]}}}}"#,
                  edited
              )
          );
          fs::write(&p, body).unwrap();
          p
      }

      #[test]
      fn test_hook_runner_honors_famp_local_root() {
          let root = TempDir::new().unwrap();
          let home = TempDir::new().unwrap();
          let bin_dir = TempDir::new().unwrap();
          let sentinel = bin_dir.path().join("famp_calls.txt");
          write_stub_famp(bin_dir.path(), &sentinel);

          // Hook row in NON-default path:
          fs::write(
              root.path().join("hooks.tsv"),
              "h1\tEdit:*.md\talice\t2026-05-03T00:00:00Z\n",
          ).unwrap();

          let edited = home.path().join("README.md");
          fs::write(&edited, "x").unwrap();
          let transcript = synth_transcript(home.path(), edited.to_str().unwrap());

          let output = std::process::Command::new("bash")
              .arg(HOOK_RUNNER)
              .env_clear()
              .env("HOME", home.path())
              .env("FAMP_LOCAL_ROOT", root.path())
              .env("PATH", format!("{}:/usr/bin:/bin", bin_dir.path().display()))
              .stdin(Stdio::piped())
              .stdout(Stdio::piped())
              .stderr(Stdio::piped())
              .spawn()
              .and_then(|mut child| {
                  use std::io::Write;
                  child.stdin.as_mut().unwrap().write_all(
                      format!(r#"{{"transcript_path":"{}"}}"#, transcript.display()).as_bytes()
                  ).unwrap();
                  child.wait_with_output()
              })
              .unwrap();

          assert!(output.status.success(), "runner must exit 0 (Stop hook contract)");

          // Sentinel proves the stub `famp` was invoked once with the right args.
          let calls = fs::read_to_string(&sentinel).unwrap_or_default();

              calls.contains("send") && calls.contains("--to") && calls.contains("alice"),
              "expected famp send --to alice fired from FAMP_LOCAL_ROOT path; got: {calls:?}"

      }

| Boundary | Description |
|----------|-------------|
| User shell env → hook-runner.sh | `FAMP_LOCAL_ROOT` is a user-controlled env var. It already governs where `scripts/famp-local hook add` writes; this plan extends the same trust boundary to the reader. |
| hook-runner.sh → `famp send` subprocess | The runner invokes `famp` from `PATH` with a constructed argv. The `--to` value is read from a TSV row controlled by the user via `famp-local hook add`. |
| Stop-hook contract | Runner MUST exit 0 on every path. `set -uo pipefail` is preserved; expansion form `${FAMP_LOCAL_ROOT:-default}` is `set -u`-safe. |
