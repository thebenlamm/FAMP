---
description: Show the FAMP inbox — recent messages received by this session.
allowed-tools: mcp__famp__famp_inbox
---

Use the `mcp__famp__famp_inbox` tool to list received envelopes since the last cursor. The list includes posts from any channels you've joined. Note: `include_terminal` is accepted for wire compatibility but is currently a no-op — broker-side terminal-FSM filtering is deferred to v1.
